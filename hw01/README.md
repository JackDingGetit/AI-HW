# 八皇后问题实现

## 实现思路

本项目使用回溯法解决八皇后问题。回溯法是一种通过探索所有可能的解来找到所有解的算法。具体实现步骤如下：

1. **定义问题**：在n×n的棋盘上放置n个皇后，使得它们不能互相攻击（即不能在同一行、同一列或同一对角线上）。

2. **回溯算法**：
   - 从第一行开始，尝试在每一列放置皇后
   - 使用`is_safe`函数检查当前位置是否安全（即不会与之前放置的皇后冲突）
   - 如果安全，继续递归处理下一行
   - 如果所有行都处理完毕，找到一个解，将其添加到结果列表中
   - 如果当前行没有安全的位置，回溯到上一行，尝试其他列

3. **安全性检查**：
   - 检查同一列是否已有皇后
   - 检查两条对角线上是否已有皇后（通过计算行差和列差的绝对值是否相等）

4. **结果表示**：使用一个列表表示棋盘，列表的第i个元素表示第i行皇后所在的列索引。

## 运行测试

### 前提条件
- 安装Python 3.x

### 执行测试
在项目根目录下运行以下命令：

```bash
python -m unittest tests/test_eight_queen.py
```

### 测试用例
- `test_n_4`：测试4皇后问题，应该有2个解
- `test_n_8`：测试8皇后问题，应该有92个解
- `test_n_1`：测试1皇后问题，应该有1个解
- `test_n_0`：测试0皇后问题，应该有0个解

## 运行示例

```python
from eight_queen import solve_n_queens, print_board

# 解决4皇后问题
solutions = solve_n_queens(4)
print(f"4皇后问题共有{len(solutions)}个解：")
for i, solution in enumerate(solutions):
    print(f"解 {i+1}:")
    print_board(solution)
```

## 性能分析

- 时间复杂度：O(N!)，其中N是皇后的数量
- 空间复杂度：O(N)，用于存储当前解和递归调用栈

对于N=8，算法可以在合理时间内找到所有92个解。
